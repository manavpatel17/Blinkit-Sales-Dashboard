# Blinkit Sales Business Intelligence Dashboard

![Dashboard Preview](images/dashboard_preview.png)

## Project Overview
An interactive Power BI dashboard built on Blinkit grocery sales data, analyzing $1.20M+ in total sales across 8,500+ items. The dashboard provides deep business insights on outlet performance, product categories, sales trends, and customer ratings.

## Dashboard Preview
The dashboard includes:
- **Total Sales:** $1.20M+
- **Average Sales:** $141
- **Total Items:** 8,523
- **Average Rating:** 3.9

## Tech Stack
- **Data Source:** Excel / CSV
- **Data Analysis:** Python, Pandas, SQL
- **Visualization:** Microsoft Power BI
- **Database Queries:** MySQL

## Key Features
- KPI Cards — Total Sales, Avg Sales, No of Items, Avg Rating
- Fat Content Analysis — Low Fat vs Regular (Donut Chart)
- Item Type Breakdown — Sales by category (Bar Chart)
- Outlet Establishment Trend — Year-wise sales growth (Line Chart)
- Outlet Size Analysis — Small, Medium, High (Donut Chart)
- Outlet Location — Tier 1, Tier 2, Tier 3 performance
- Outlet Type Table — Detailed comparison of all outlet types

## Key Insights
- Supermarket Type 1 generated highest sales of $787.55K
- Tier 3 locations contributed maximum sales of $472.13K
- Fruits and Snack Foods are top-selling categories
- Regular fat content products outsell Low Fat products
